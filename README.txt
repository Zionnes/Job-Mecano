Pour l'installé : 
-téléchargé en ZIP
-Glisser le dans votre fichiers ressources 
-Crée une catégorie qui s'appelle [JOB]
-Mettez dans votre server.cfg "ensure esx_lscustom" (Si vous etes sur un hébergeur) et "ensure esx_mecanojob"
-Mettez dans votre server.cfg "start esx_lscustom" (Si vous etes en localhost) et "start esx_mecanojob
